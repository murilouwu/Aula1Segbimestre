Atividade sobre Mobile:

1- Diferença entre Nativo e Híbrido:

Nativo: Desenvolvimento específico para cada plataforma (iOS, Android), usando linguagens como Swift (iOS) e Kotlin (Android).

Híbrido: Uso de tecnologias web (HTML, CSS, JavaScript) para criar apps que podem rodar em múltiplas plataformas usando frameworks como React Native e Flutter.

2- Desvantagens do Nativo:

- Demanda tempo e recursos para desenvolver e manter duas versões do app (iOS e Android).

- Necessidade de aprender linguagens diferentes para cada plataforma.

3- Progressive Web App (PWA):

- PWA é um tipo de aplicativo que utiliza tecnologias web para oferecer uma experiência semelhante a apps nativos.

- Ele pode ser acessado através de navegadores e instalado na tela inicial do dispositivo.

4- Linguagens nativas para Android:

  • Kotlin
  • Java
  • C++
  • Python

4- Linguagens nativas para iOS:

  • Swift
  • Objective-C

5- Publicação nas lojas:

- Para apps nativos, é necessário criar pacotes específicos para cada loja (App Store e Google Play) e seguir as diretrizes de cada plataforma.

- Para PWAs, não é necessário publicar em lojas; eles são acessados através de navegadores.

6- Plataformas híbridas de desenvolvimento:

  • React Native
  • Flutter

7- Vantagens de usar plataforma híbrida:

- Desenvolvimento mais rápido e econômico, pois um código base é usado para várias plataformas.

- Maior flexibilidade ao compartilhar recursos entre apps iOS e Android.

8- Hot Reload e Live Reload:

Hot Reload: Recarrega apenas as partes modificadas do código durante o desenvolvimento, mantendo o estado da aplicação.

Live Reload: Recarrega toda a aplicação quando ocorrem mudanças no código, reiniciando o estado da aplicação.
